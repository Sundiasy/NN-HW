# Homework 1 Report

## 1. 数据集

- 数据集名称：Concrete Compressive Strength
- 样本数：1030
- 输入特征数：8
- 输出变量：Concrete compressive strength (MPa)

## 2. 特征分析

在训练集上计算 Pearson 相关系数，并导出排序结果与条形图。该分析用于观察每个输入变量与抗压强度之间的线性相关程度。

## 3. 数据预处理

- 按题目要求，将前 80% 样本作为训练集，后 20% 样本作为测试集。
- 对输入特征使用训练集统计量进行标准化。
- 为了稳定 MLP 训练过程，对目标值使用 `log1p` 后再做标准化，并在预测时还原到原始 MPa 空间。

## 4. 模型设计

- 框架：PyTorch
- 模型：MLP
- 结构：`8 -> 64 -> 32 -> 16 -> 1`
- 激活函数：ReLU
- 优化器：AdamW
- 损失函数：MSELoss
- 训练策略：从训练集尾部划出验证集，使用早停选择最佳模型

## 5. 评估方式

- 主指标：测试集均方误差（MSE）
- 可视化：训练/验证损失曲线、测试集真实值与预测值散点图

## 6. 结果

- 测试集 MSE：`76.9802`
- 测试集 R²：`0.4957`
- 最优轮次：`665`

相关性排序前三的特征如下：

- `Cement`：`0.4696`
- `Superplasticizer`：`0.4614`
- `Age`：`0.3403`

作为参考，线性回归基线在同一测试集上的 MSE 为 `75.7221`，R² 为 `0.5039`。说明在当前按顺序划分的数据设置下，简单线性模型与 MLP 的表现较为接近；MLP 已达到与基线相近的回归效果，并完成了题目要求的神经网络实现、训练和可视化分析。
